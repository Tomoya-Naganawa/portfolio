　概要
アプリ名：Movie salon
URL：https://www.moviesalon.site
GitHub：https://github.com/Tomoya-Naganawa/movie_salon
アプリの説明：
Movie salonは映画のレビューを投稿し、それに対するコメントやいいねができる映画評論アプリです。映画の情報を提供するTMDb APIのデータを利用しており、映画のあらすじや出演俳優、公開日などの情報を検索することができます。
※ゲストとしてログインできます
　メールアドレス：guest1@guest.com
　パスワード：guest1234
　機能
■ユーザ関連機能
・アカウント作成、編集
・ログイン・ログアウト
・メールアドレス変更

■メイン機能
・映画検索、詳細表示(TMDB APIからの映画データの受け取り)
・映画データDBへの保存
・映画の評価の集計
・映画のキーワード登録
・レビュー投稿、編集、削除
・レビューに対するコメント投稿、削除
・レビューに対するいいね登録、解除
・レビューの閲覧数ランキング
・レビューのソート
　技術要素
・開発環境：Dockerコンテナ
 - コンテナ管理：docker-compose + Bashシェルスクリプト
 - webサーバ：Nginx + PHP-FPM
 - DBサーバ：MySQL, Redis
・本番環境：Heroku
・CI/CDツール：CircleCI
・言語：PHP7.3
・フレームワーク：Laravel5.8
・データベース：MySQL5.7, Redis
・データベースGUI管理ツール：phpMyAdmin
・ファイルストレージ：AWS S3
・フロントエンド：Laravel, Vue.js, Bootstrap
・外部API：Facebookログイン, Googleログイン, TMDb
・HTTPクライアント：Guzzle

■Laravel
・DBリレーション
・DBトランザクション
・ページネーション
・バリデーション：Validator
・ソーシャルログイン：Socialite
・閲覧数ランキング
 - キャッシュ管理・操作：Redis
・機能テスト：PHPunit
 - テストデータ生成：Seeder, Factory, UploadedFile, dataProvider
 - モック作成：Mockery
・S3連携：flysystem-aws-s3-v3
・外部APIへのHTTPリクエスト：Guzzle
